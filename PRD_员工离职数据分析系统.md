## 1. 产品概述

### 1.1 产品定位

员工离职数据分析系统是一款基于Web的数据分析工具，帮助HR部门和管理层快速了解员工离职情况，识别风险部门，提供数据驱动的决策支持。

### 1.2 核心价值

- **数据驱动**：通过Excel导入，自动计算离职率等核心指标
- **可视化分析**：多种图表类型，直观展示离职分布
- **AI智能洞察**：集成大模型，自动生成深度分析报告
- **便捷导出**：支持Excel和PDF格式报告导出

## 2. 功能需求

### 2.1 数据导入

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| Excel文件上传 | 支持拖拽上传.xlsx/.xls格式 | P0 |
| 多Sheet识别 | 自动识别期初人数、期末人数、离职数据Sheet | P0 |
| 月份自动关联 | 根据Sheet名称推断月份（1月、2月等） | P0 |
| 数据清洗 | 自动处理空值、格式异常 | P1 |

### 2.2 离职率计算

**计算公式**：
```
月度离职率 = 离职人数 / 平均人数 × 100%
平均人数 = (期初人数 + 期末人数) / 2
```

### 2.3 数据可视化分析

#### 2.3.1 各部门离职情况（合并图表）

**图表类型**：组合图表（纵向柱状图 + 折线图）

| 元素 | 说明 |
|------|------|
| 柱状图 | 显示各部门离职人数（深蓝色） |
| 折线图 | 显示各部门离职率（红色） |
| X轴 | 部门名称（倾斜排列） |
| 左Y轴 | 离职人数 |
| 右Y轴 | 离职率(%) |

**交互特性**：
- 自动过滤离职人数为0的部门
- 按离职率降序排列
- 悬停显示详细数据

#### 2.3.2 离职类型与司龄分布

| 图表 | 类型 | 说明 |
|------|------|------|
| 离职类型分布 | 饼图 | 显示主动离职/被动离职等占比 |
| 司龄分布 | 饼图 | 显示0-0.5年/0.5-1年/1-3年等分布 |

#### 2.3.3 离职原因与职级分布

| 图表 | 类型 | 说明 |
|------|------|------|
| 离职原因分布 | 纵向柱状图 | 按人数降序排列，最多显示10项 |
| 职级分布 | 纵向柱状图 | 按人数降序排列 |

#### 2.3.4 月度趋势分析

| 图表 | 类型 | 说明 |
|------|------|------|
| 离职率趋势 | 组合图表 | 折线图显示离职率，柱状图显示离职人数 |

### 2.4 AI智能分析

| 功能点 | 描述 |
|--------|------|
| AI分析报告 | 调用Groq大模型，自动生成深度洞察 |
| 风险预警 | 识别离职率异常高/低的部门 |
| 优化建议 | 提供具体的改进建议 |

### 2.5 数据导出

| 格式 | 内容 |
|------|------|
| Excel | 月度统计、部门分析、离职明细 |
| PDF | 完整分析报告（含AI分析内容） |

## 3. 可视化规范

### 3.1 图表配色方案

| 用途 | 颜色代码 | 说明 |
|------|----------|------|
| 主色 | #4F46E5 | 靛蓝色，用于离职人数柱状图 |
| 危险色 | #EF4444 | 红色，用于离职率折线图 |
| 强调色 | #F59E0B | 琥珀色，用于离职原因图表 |
| 辅助色 | #8B5CF6 | 紫色，用于职级分布图表 |
| 成功色 | #10B981 | 绿色，用于正向指标 |
| 背景色 | #FFFFFF | 白色图表背景 |

### 3.2 图表布局规范

| 元素 | 规范 |
|------|------|
| 图表高度 | 350-400px |
| 标题位置 | 居中，16号字体 |
| 图例位置 | 顶部居中，横向排列 |
| 网格线 | 浅灰色，水平方向 |
| 数据标签 | 柱子外侧显示 |

### 3.3 数据过滤规则

- **离职人数为0的部门**：不显示在图表中
- **离职原因**：最多显示Top 10
- **排序规则**：默认按离职率/人数降序排列
